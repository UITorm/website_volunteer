# 网站访问
https://wennuangangcheng.netlify.app/

# 本地化部署网站
1. 从Releases中下载源代码（windows选择zip，Linux/MacOS选择tar.gz）
2. 打开index.html文件即可

# 主要项目文件介绍
- **index.html**: 主页面渲染；实现页面的点击跳转（比如：点击“怎么查天气预报”，会跳转至phone-guide.html）
- **phone-guide.html**: 板块一“手机使用指引”
- **fraud-guide.html**: 板块一“防骗指南”
- **health-guide.html**: 板块一“健康管理”
- **shopping-guide.html**: 板块二“买菜指引”
- **medical-guide.html**: 板块二“就医指引”
- **travel-guide.html**: 板块二“出行指引”
- **fedback.html**: 板块三“网站评价与建议”
